## Build Instructions

Compile the project:

```bash
mvn compile
```

Run tests:

```bash
mvn test
```

Package the project:

```bash
mvn package
```

Install the project to the local Maven repository:

```bash
mvn install
```

Run the executable JAR:

```bash
java -jar target/hello-world-1.0-SNAPSHOT.jar
```

Alternatively, run using the classpath:

```bash
java -cp target/hello-world-1.0-SNAPSHOT.jar com.mycompany.app.App
